# Folder or File Hiding in Permanent Way

## To Hide a Folder/File
01.	 Open cmd and go to the file/folder directory.
* `cd C:\Users\AIIH`
02.	To hide pictures
* `attrib +h +s Pictures`

## To Unhide the Folder/File
01.	 Open cmd and go to the file/folder directory.
* `cd C:\Users\AIIH`
02.	To unhide pictures
* `attrib -h -s Pictures`
